# Soom Shop
## Start Project
```
git clone https://github.com/soominpark9/Shop.git
cd online/
npm i
npm start 
```

## Stack
- React
- Redux-toolkit
- SCSS

## Directory Structure
```
online/ # react app
.gitignore
```

## REST API
- `GET /MainPage` - show all products
- `GET /EachDetail` - show each bed data
- `DELETE /Cart` - remove a specific bed
